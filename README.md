GitPet is a VS Code extension that helps you stay productive by taking care of a virtual pet. The pet grows with you as you code, and you can interact with it through a simple command palette. It's a fun way to stay motivated and keep your coding habits in check.

Features:

- Virtual pet that grows with you as you code
- Interactive command palette for easy access to features
- Fun and engaging way to stay motivated
- Track your coding progress and habits
- Keep your coding habits in check
